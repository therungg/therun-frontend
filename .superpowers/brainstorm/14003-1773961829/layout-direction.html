About Page Layout

Pick the structure that fits best for a feature showcase

A

Feature Grid

Hero tagline at top, then a grid of feature cards (icon + title + short description). Clean, scannable, modern. Think product landing page without the marketing fluff.

B

Sectioned Showcase

Group features into logical sections (Track, Compete, Analyze, etc.) with alternating left/right layouts. Each section gets a heading, brief description, and a few bullet points. More structured, gives hierarchy.

C

Hybrid

Short hero tagline, then sectioned groups where each group contains a small grid of feature cards. Best of both — scannable cards with logical grouping.
